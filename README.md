# Binance Funding Rate Monitor Bot

Monitors Binance funding rates and alerts on high/low rates.

## Commands

- `/subscribe` - Subscribe to alerts
- `/unsubscribe` - Unsubscribe
- `/rates` - Show top funding rates
- `/status` - Status

## Env

See `.env.example`.

## Deploy

Use `Dockerfile` and `render.yaml` for Render.com.
